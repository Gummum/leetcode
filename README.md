## 回溯

``` c
void backtracking(参数) {
    if (终止条件) {
        存放结果;
        return;
    }

    for (选择：本层集合中元素（树中节点孩子的数量就是集合的大小）) {
        处理节点;
        backtracking(路径，选择列表); // 递归
        回溯，撤销处理结果
    }
}
```

## 动态规划


### 思路1

1. 定义数组元素的含义，一般是要求的答案。
2. 找出数组元素之间的关系式：dp[n] 一定会和 dp[n-1], dp[n-2]....存在某种关系的。
3. 找出初始值，注意特殊值。

### 思路2

1. 确定dp数组（dp table）以及下标的含义
2. 确定递推公式
3. dp数组如何初始化
4. 确定遍历顺序
5. 举例推导dp数组
